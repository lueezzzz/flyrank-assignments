# Flyrank AI: Backend Engineering Internship

## Overview
This repository is a compilation of the projects and tasks I completed during my Backend Engineering Internship at Flyrank AI. It serves as a record of my work and showcases my progression in building, structuring, and containerizing modern application architectures.

## Repository Structure
* **Project 1: Simple FastAPI CRUD Backend w React + Vite Frontend** 
  * **Tech Stack:** Python, FastAPI, React, Vite, TypeScript, Docker, Docker Compose
  * Built a foundational REST API supporting Create, Read, Update, and Delete operations.
  * Implemented modular routing, Pydantic schemas for data validation, and CORS middleware for frontend communication.

## Getting Started
The projects in this repository are containerized to ensure a consistent development environment. You do not need to install local dependencies to run the code.

To spin up the environment, navigate to the root directory and execute:

```bash
docker compose up --build
```
